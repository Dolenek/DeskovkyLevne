# Deskovky Levne

Slug-first board-game price tracking app (React frontend + Go read API).

## Quick Start
Frontend and local API:
```bash
npm install
npm run dev
```

Frontend only:
```bash
npm run dev:frontend
```

Backend API only:
```bash
npm run api:dev
```

Tests:
```bash
npm test
```
